VS Code

    This page addresses setting up live coding in VS Code, so you can see the changes you make to your page as you make them.
  1. To set up live coding in VS Code install the Live Server Extension.
  2. Manage your extensions by typing 'ext' into the address bar at the top of VS code.
  3. Type Live Server into the Search bar.
  4. Select the Live Server extension. {hover here}
  5. I also reccomend the extension Prettier for formatting and reference assistance. {hover here}
